# 한량낚시

월령, 기상, 수온 추정, 시간대별 피딩 확률을 보여주는 민물낚시 출조 예측 웹앱입니다.

## 주요 수정 사항

- 오늘 날짜: 현재기온, 현재기압, 현재풍속, 현재강수 우선 표시
- 과거 날짜: 당시 평균기온, 평균기압, 평균풍속, 총강수 표시
- 미래 날짜: 예상 평균기온, 예상 평균기압, 예상풍속, 예상강수 표시
- 수온: 실측 API 시도 후 실패 시 최근 7일 평균기온 기반 추정
- 앱 설치명: 한량낚시
